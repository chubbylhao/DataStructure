<h2 align = "center">数据结构基础</h2>

#### 线性表

- ##### 线性表的顺序存储结构（顺序表）

|      函数      |                功能                | 时间复杂度 |
| :------------: | :--------------------------------: | :--------: |
| 构造和析构函数 |               ......               |  **O(1)**  |
|     Length     |          返回顺序表的表长          |  **O(1)**  |
|      Get       |  按位查找，返回某个位置处元素的值  |  **O(1)**  |
|     Locate     |     按值查找，返回某个值的位置     |  **O(n)**  |
|     Insert     | 插入操作，在某个位置上插入某个元素 |  **O(n)**  |
|     Delete     |   删除操作，删除某个位置处的元素   |  **O(n)**  |
|   PrintList    |  遍历操作，打印顺序表中的所有元素  |  **O(n)**  |
